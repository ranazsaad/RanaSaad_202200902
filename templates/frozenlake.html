Quick Start Guide
  1. Select an algorithm from above
  2. Adjust parameters if needed
  3. Toggle slippery surface
  4. Click "Train" to start training
  5. Click "Test Policy" to see results
How It Works
  • Agent starts at S (top-left) and must reach G (bottom-right)
  • Agent receives +10.0 reward for reaching goal
  • Falling in a hole gives -10.0 reward
  • Each step costs -0.1 reward
  • With slippery surface, actions have 33% chance to slip to random direction
Algorithms
  • Q-Learning: Off-policy, model-free TD
  • SARSA: On-policy, model-free TD
  • Value Iteration: Model-based DP
  • Policy Iteration: Model-based DP
  • Monte Carlo: Model-free, learns from episodes
  • TD Learning: Temporal Difference learning
  • N-step TD: Multi-step temporal difference
